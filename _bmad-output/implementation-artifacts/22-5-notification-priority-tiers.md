# Story 22.5: Notification Priority Tiers

Status: done

## Story
As a **user**, I want notifications in 3 tiers with different visual treatment, So that critical alerts stand out.

## Acceptance Criteria
1. Tier 1 (Red): agent stuck, conflict, decision needed — prominent alert
2. Tier 2 (Amber): PR ready, scope creep — badge indicator
3. Tier 3 (Green): story completed, milestone — subtle toast
4. Tier assignment is rule-based and configurable
